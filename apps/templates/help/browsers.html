To be able to play you need a modern web browser that supports WebSockets or Flash. If you run something like Chrome or Safari it should be able to use WebSockets. If you use something like Firefox, Opera or one of the latest Internet Explorer you need to have Flash support.

Why not IE6?

Kwissle uses modern web-sockets technology to ensure an almost instantaneous quiz battle environment - older browsers simply don't support this technology.
